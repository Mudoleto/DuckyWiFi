This URL address is for downloading the digistump library to use digispark devices follow the steps:

From the arduino IDE go to the navigation bar and use:
├──File
└──Preferencesi

In the section "Additional board gesture URLs" section, paste the link at the end of this text file.
│  
│  
└──>https://raw.githubusercontent.com/digistump/arduino-boards-index/master/package_digistump_index.json

Now, you must go to "Tools". then "Boards" and at the end "Boards manager" and type in the type section: "Esp32" and install the one named "ArduinoEsp32 boards".

And that's it, you can use your Esp32 board.
